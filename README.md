# Amazon ML Challenge 2025 – Price Prediction Pipeline

🏆 **Final Rank:** **312 / 85,000+ Participants** (Top **0.37%**)

An end-to-end machine learning solution developed for the **Amazon ML Challenge 2025**, focusing on accurate product price prediction through advanced feature engineering, ensemble learning, and multimodal feature extraction.

---

## 📌 Overview

This project presents a complete machine learning pipeline for predicting product prices using structured data, textual descriptions, and image-derived features.

The solution combines multiple feature engineering techniques with ensemble learning models to improve prediction performance on the competition leaderboard.

---

## 🚀 Features

- Data preprocessing and cleaning
- Feature engineering on structured attributes
- TF-IDF vectorization for textual features
- Dimensionality reduction using Truncated SVD
- Target Encoding for categorical variables
- Image feature extraction using OpenCV
- LightGBM model training
- Ridge Regression model
- Ensemble learning
- K-Fold Cross Validation
- Prediction generation for competition submission
